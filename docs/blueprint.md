# **App Name**: Global Opportunity Tracker

## Core Features:

- Attendance Tracking: Mark student attendance as present, late, or excused, with reasons.
- QR Code Check-in: Allow students to check in by scanning a QR code. Allow admin or teacher to set status manually.
- Attendance Calendar: Visualize attendance data on a calendar to identify patterns and trends.
- Analytics Dashboard: Provide an overview of attendance metrics.
- Automated Late Check: Automatically mark students late if they have not checked in after a set time. Use expected time as a tool for determining lateness.
- Report Generation: Generate attendance reports that can be exported as CSV or PDF.
- User Role Management: Admin, teacher, parent and student roles with role-specific access and features.

## Style Guidelines:

- Primary color: Light desaturated purple (#D8CEE6), echoing the colors used in the provided HTML, lending a calm and studious atmosphere.
- Background color: Very light purple (#F7F5FB).
- Accent color: Deep violet (#4F46E5) for interactive elements and key information.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet welcoming feel.
- Use consistent, professional, and simple icons to represent actions and status, aiding quick recognition.
- Employ a clean, grid-based layout for all the dashboards, ensuring information is easily accessible and well-organized.
- Subtle animations and transitions on UI elements to provide feedback and a polished user experience.
